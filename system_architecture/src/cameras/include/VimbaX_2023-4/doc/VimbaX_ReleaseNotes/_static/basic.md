# Sphinx Basic Theme CSS Stylesheet

This is the basic.css stylesheet from the Sphinx documentation generator, version 2007-2023, used as part of the VimbaX 2023-4 SDK release notes documentation. The stylesheet provides comprehensive styling for Sphinx-generated documentation pages, including main layout structure with sidebar navigation, related links bar, searchbox styling, body content formatting, tables (including docutils tables, index tables, and content tables), figures, field lists, code blocks with syntax highlighting support, admonitions (notes, warnings, etc.), footnotes, citations, glossary definitions, math display, and print-specific styles. It also includes specific styling for C/C++ function signatures with syntax coloring for keywords, literals, and strings.

**Key concepts:** `Sphinx documentation styling`, `Page layout with sidebar navigation`, `Documentation body styles`, `Code highlighting and display`, `Table and figure formatting`, `Admonitions and topic boxes`, `Search page styling`, `Print media queries`, `Responsive typography`, `C++ code signature styling`

## Sections

- **Main Layout** — Basic layout styles for document structure including clearers and sections
- **Related Bar** — Styling for navigation breadcrumb/related links bar
- **Sidebar** — Sphinx sidebar wrapper and searchbox styling
- **Search Page** — Search results page styling
- **Index Pages** — General and module index table styling
- **Body Styles** — Main content body typography and alignment
- **Sidebars and Topics** — Floating sidebar boxes and topic containers
- **Admonitions** — Note, warning, and other admonition box styles
- **Tables** — Docutils tables, citations, and field lists
- **Figures** — Figure and caption styling
- **Object Descriptions** — API documentation signatures with C/C++ specific coloring
- **List Styles** — Ordered and unordered list formatting
- **Code Displays** — Code block and syntax highlighting styles
- **Math Display** — Mathematical notation styling
- **Print Stylesheet** — Media query for print output
