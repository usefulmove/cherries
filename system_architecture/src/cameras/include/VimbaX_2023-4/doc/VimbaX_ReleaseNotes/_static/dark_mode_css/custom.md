# VimbaX Dark Mode Custom CSS Styles

This CSS file provides custom styling for the VimbaX Release Notes documentation, specifically to support dark mode functionality. It includes a fixed-position theme-switcher button with circular design and shadow effects, removes box shadows from input fields, sets link colors in the navigation content, and applies CSS transitions (0.2-0.3 second ease-in-out) to various page elements including body, navigation, code blocks, and definition terms for smooth theme switching. The styles are designed to work with the Read the Docs (Sphinx) theme framework.

**Key concepts:** `theme-switcher button`, `dark mode transitions`, `Read the Docs theme customization`, `input field styling`, `navigation styling`

## Sections

- **Input Field Styling** — Removes box shadows and adds border radius to search inputs
- **Theme Switcher Button** — Fixed-position circular button for toggling dark/light mode
- **Link Colors** — Sets navigation content link colors to blue
- **Transition Effects** — Applies smooth transitions to multiple page elements for theme switching
