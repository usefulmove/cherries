# Vimba X for macOS Release Notes (2023-4)

This HTML documentation file contains the official release notes for Vimba X SDK version 2023-4 for macOS by Allied Vision. It marks the first release with macOS support, targeting Mac mini and MacBookPro systems with M2 chips running macOS Sonoma 14.1.1. The document details the supported components including GigE and USB Transport Layers, VmbC/VmbCPP/VmbPy APIs, Image Transform Library, Vimba X Viewer, and Firmware Updater tools. It provides comprehensive information about known issues related to camera detection, Vimba X Viewer behavior, API limitations, and example code quirks, along with workarounds. Installation instructions explain the .dmg-based installation process. The release history section documents changes across versions from the initial 2022-1 release through 2023-4, tracking the evolution of macOS support and feature additions like event support and sequencer tabs.

**Key concepts:** `First macOS support release`, `Supported hardware: Mac mini and MacBookPro with M2 chip`, `macOS Sonoma 14.1.1 compatibility`, `GigE Transport Layer v1.9.5`, `USB Transport Layer v1.4.6`, `VmbC/VmbCPP/VmbPy APIs v1.0.4-1.0.5`, `Vimba X Viewer v1.3.0`, `Known issues and workarounds`, `Installation via .dmg and .pkg files`, `Release history from 2022-1 to 2023-4`
