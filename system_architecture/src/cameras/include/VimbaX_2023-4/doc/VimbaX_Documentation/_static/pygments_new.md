# Pygments Syntax Highlighting CSS Theme

This CSS file provides a complete Pygments syntax highlighting theme for the VimbaX 2023-4 documentation. It defines styling rules for all major token types including comments (green italic), keywords (green bold), strings (red/brown), numbers (gray), class/function names (blue), operators, and various other syntax elements. The theme uses a light gray background (#f8f8f8) with carefully chosen colors to provide readable code highlighting that is typical of documentation systems built with Sphinx.

**Key concepts:** `Pygments token classes`, `Syntax highlighting color scheme`, `Code block styling`, `Comment styles (green/italic)`, `Keyword styles (green/bold)`, `String literals (red/brown)`, `Class and function names (blue)`, `Number literals (gray)`

## Sections

- **Base styles** — Highlight background and line highlighting
- **Comments** — Styling for various comment types (single, multi, hashbang, etc.)
- **Keywords** — Styling for keywords, constants, declarations, types
- **Literals** — Styling for strings and numbers
- **Names** — Styling for classes, functions, variables, decorators, etc.
- **Operators** — Styling for operators and whitespace
- **Generic tokens** — Styling for diffs, errors, headings, prompts
