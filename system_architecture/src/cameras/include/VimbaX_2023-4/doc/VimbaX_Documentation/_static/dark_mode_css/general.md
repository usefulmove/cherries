# VimbaX Documentation Dark Mode General CSS

This CSS file is part of the VimbaX 2023-4 camera SDK documentation's dark mode implementation. It defines styles for a circular theme switcher button positioned at the bottom-right corner of the page, removes box shadows from various input types, and applies smooth CSS transitions to numerous documentation elements including the navigation sidebar, content areas, code blocks, and definition lists. The selectors target elements specific to the Sphinx/ReadTheDocs documentation theme (wy-* classes, rst-content).

**Key concepts:** `input field styling`, `theme switcher button`, `CSS transitions`, `ReadTheDocs/Sphinx theme overrides`, `dark mode UI component`

## Sections

- **Input field reset** — Removes box shadows from various HTML input types
- **Theme switcher button** — Styles for the fixed position circular dark/light mode toggle button
- **Transition definitions** — Applies smooth transitions to various documentation layout elements
