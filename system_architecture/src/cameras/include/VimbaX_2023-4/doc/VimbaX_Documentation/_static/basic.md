# Sphinx Basic Theme CSS Stylesheet

This is the basic.css stylesheet from Sphinx 2023-4, used as part of the VimbaX SDK documentation. It provides the foundational CSS rules for the Sphinx documentation system's basic theme. The stylesheet covers comprehensive styling for documentation structure including main layout with sidebars, navigation bars, search functionality, index pages, body content, tables, figures, field lists, code displays with syntax highlighting, admonitions (warnings, notes, etc.), topics, glossaries, math displays, and print media support.
