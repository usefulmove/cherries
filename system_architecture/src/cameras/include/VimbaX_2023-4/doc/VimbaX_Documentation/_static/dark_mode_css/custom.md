# VimbaX Documentation Dark Mode Custom CSS

This CSS file provides custom styling for a dark mode feature in the VimbaX SDK documentation. It defines styles for a circular theme-switcher button fixed at the bottom-right of the screen with shadow effects, removes box shadows from form inputs, and sets up CSS transitions for smooth theme switching across various documentation elements including navigation, content areas, code blocks, and definition lists. The styling targets the Read the Docs (Sphinx) theme framework.

**Key concepts:** `theme switcher button`, `dark mode styling`, `Read the Docs theme customization`, `CSS transitions`, `form input styling`

## Sections

- **Input styling** — Removes box shadows from form input elements
- **Theme switcher button** — Fixed circular button for toggling dark/light mode
- **Link styling** — Custom link colors for navigation content
- **Transition definitions** — CSS transitions for smooth theme switching across multiple elements
