# Sphinx Documentation Build Info

This is a Sphinx documentation system build information file (.buildinfo) for the VimbaX 2023-4 documentation. It stores hashes of the configuration and tags used during the documentation build process. These hashes are used by Sphinx to determine whether a full rebuild is necessary when regenerating documentation - if this file is missing or the hashes don't match the current configuration, Sphinx will perform a complete rebuild rather than an incremental one.

**Key concepts:** `Sphinx build info version 1`, `configuration hash`, `tags hash`, `incremental build tracking`
