# GNU Lesser General Public License v3.0 (LGPL-3.0)

This HTML file contains the complete text of the GNU Lesser General Public License version 3.0 (LGPL-3.0), dated June 29, 2007, published by the Free Software Foundation. The LGPL is designed primarily for software libraries and permits proprietary software to link with LGPL-licensed libraries while maintaining the freedom of the library itself. The license covers key concepts including definitions of Library, Application, and Combined Works, rules for conveying modified versions, object code incorporating library header files, combined works requirements, combined libraries provisions, and revised versions of the license.
