# VmbCPP Main Header

This is the main umbrella header file for the VmbCPP (Vimba C++ API) library from Allied Vision Technologies, part of the VimbaX 2023-4 SDK. It serves as a convenience header that includes all the essential components of the SDK including Camera, Interface, VmbSystem, FeatureContainer, and various observer interfaces (ICameraFactory, ICameraListObserver, IInterfaceListObserver, IFeatureObserver, IFrameObserver) as well as Frame handling. This allows users to include a single header to get access to the complete VmbCPP API functionality for interfacing with Allied Vision cameras.

**Key concepts:** `VmbCPP API`, `Camera interface`, `Frame observation`, `Feature containers`, `Camera list observers`
