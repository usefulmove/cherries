# MAP.md — Project: src

This document serves as the top-level navigation map for a ROS 2 monorepo implementing an automated cherry-sorting system. It provides an overview of all major subsystem directories including camera drivers (Allied Vision Vimba SDK and Cognex), cherry detection with ConvNeXt V2 ML models, FANUC robot and EtherNet/IP PLC communications, Qt-based GUI, tracking projector, and various supporting services. The map directs agents to follow INDEX.md links within each directory for detailed exploration of individual packages and files.

**Key concepts:** `ROS 2 monorepo structure`, `cherry detection and tracking`, `camera drivers (Allied Vision/Cognex)`, `PLC/robot communications (FANUC, EtherNet/IP)`, `HDR image acquisition`, `system navigation via INDEX.md files`
