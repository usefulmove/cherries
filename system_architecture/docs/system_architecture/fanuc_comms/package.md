# fanuc_comms ROS 2 Package Manifest

This document describes the ROS 2 package manifest for fanuc_comms, a Python-based ROS 2 package designed for communication with Fanuc robots. It uses package format 3 and the ament_python build type, indicating a pure-Python ROS 2 package. The package is maintained by Wesley Havener at Saber Engineering and is at an early development stage (version 0.0.0), with placeholder descriptions and license fields. The declared dependencies are standard ROS 2 test tools including ament_copyright, ament_flake8, ament_pep257, and python3-pytest for code style and licensing compliance checks.
